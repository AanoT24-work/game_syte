Тема проекта
Разработка веб-приложения социальной сети с функционалом публикации постов, комментариев, лайков и встроенным мессенджером с поддержкой редактирования сообщений.

Основные сущности

1. Пользователь (User)
Логин и пароль с хэшированием
Аватар пользователя
Статус (user/admin)
Дата регистрации
Связи: посты, комментарии, отправленные/полученные сообщения

2. Пост (Post)
Текст контента
Изображение (опционально)
Дата создания
Список пользователей, поставивших лайк
Связь с автором и комментариями

3. Комментарий (Comment)
Текст комментария
Дата создания
Связь с пользователем и постом

4. Сообщение (Message)
Текст сообщения
Отправитель и получатель
Время отправки и прочтения
Флаг редактирования
История изменений (до 10 последних правок)
